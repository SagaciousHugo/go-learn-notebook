# 前言

记录一下自己go语言学习的笔记，希望可以达到以下几个目标

* 详尽的基础语法----可作为字典备查
* 准确、简单易懂的解释----可作为准备面试的复习笔记
* 给出部分源码定义----加深对Go语言的理解

每一部分笔记分为以下几部分

* 基础概念或语法详细介绍
* 原理、源码解析
* 常见标准用法
* 开源库中的应用
* 复习题
